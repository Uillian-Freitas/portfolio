# 🖥️ Portfólio — Uillian Freitas

Portfólio pessoal desenvolvido para apresentar projetos, habilidades e tecnologias utilizadas durante minha jornada como **Desenvolvedor Full Stack Java**.

O site possui uma estética inspirada em **terminal hacker / cyberpunk**, com animações em JavaScript, efeito Matrix no fundo e interface baseada em comandos de terminal.

---

🔗 **Repositório:**
https://github.com/Uillian-Freitas/portifolio

---

# 🎯 Objetivo do Projeto

Criar um portfólio moderno e interativo para apresentar:

* projetos desenvolvidos
* tecnologias dominadas
* experiência prática com desenvolvimento front-end
* identidade visual única

Além de consolidar conhecimentos em **HTML, CSS e JavaScript puro** sem utilização de frameworks.

---

# 🧠 Conceitos Aplicados

✔ Estruturação semântica com **HTML5**
✔ Estilização avançada com **CSS3**
✔ Layout responsivo com **Flexbox e Grid**
✔ Manipulação do DOM com **JavaScript Vanilla**
✔ Animações e efeitos visuais
✔ Intersection Observer para animações on-scroll
✔ Canvas API para efeito **Matrix background**
✔ Menu responsivo com **hamburger mobile**
✔ Efeito **typing animation** estilo terminal
✔ Componentização visual em seções

---
